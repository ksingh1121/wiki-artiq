# wiki-artiq

ARTIQ experiment control wiki — hosted at https://ksingh1121.github.io/wiki-artiq

Built with [Jekyll](https://jekyllrb.com/) and the [Just the Docs](https://just-the-docs.com/) theme, deployed via GitHub Pages.

## Adding content

1. Create a `.md` file in the appropriate `docs/` subfolder.
2. Add the frontmatter block at the top (copy from an existing file).
3. Commit and push — GitHub Pages rebuilds automatically.

## Structure

```
docs/
├── getting-started/
├── hardware/
├── experiments/
└── reference/
```
